# Purpose

This is second version of [Expense Assistant](https://github.com/zoeyzou/expense-assistant) for have a taste of Redux and some other libraries.

# Stack & Tools

- React
- Typescript
- Redux & Thunk
- React Table
- Webpack
- Babel
- Styled-components
- Prettier

# Get started

- Prepare backend by cloning [this](https://github.com/zoeyzou/frontend-challenge.git)
- run backend server
- Clone this repo
- run `npm i`
- run `npm start`

You are ready to go.

# Note

It's in working progress, so you would see some major and some minor issues.
